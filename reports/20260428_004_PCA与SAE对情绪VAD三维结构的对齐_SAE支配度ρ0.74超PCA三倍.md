# PCA 与 SAE 对情绪 VAD 三维结构的对齐 — SAE 支配度 ρ=0.74，超 PCA 三倍

## 背景与目标

**问题**：PCA 找方差最大方向，PC1 混合了效价与唤醒度，PC2 混合了唤醒度与支配度，且没有任何 PC 能干净对齐支配度（最高 ρ=0.34）。

**假设**：SAE 的稀疏约束将残差流的 superposition 分解成更单义的特征，再用 NRC-VAD 标签选取相关特征合成方向，应能比 PCA 更干净地分离三个 VAD 维度。

**修正点（相比 EMI-003）**：
- 叙述数量从 3 条增至 20 条
- 编码顺序从 mean-then-encode 改为 encode-then-mean（逐叙述过 SAE，再在特征空间取均值）

---

## 方法

### 数据准备

```
narrative_matrix (171, 20, 2304)
    每条叙述独立过模型取 token 均值
    ↓
emotion_matrix (171, 2304) = narrative_matrix.mean(axis=1)   # 用于 PCA
```

### PCA

StandardScaler 标准化 → PCA(n_components=10) → 计算 PC1-10 与 VAD 三维的 Spearman ρ。

### SAE 方向构造（半监督）

```
# 逐叙述编码（encode-then-mean）
features_3d (171, 20, 16384) = JumpReLU(narrative_matrix @ W_enc + b_enc)
features (171, 16384) = features_3d.mean(axis=1)

# VAD 方向合成（以效价为例）
ρⱼ = spearmanr(features[:, j], valence_scores)   # 对 156 个匹配情绪
direction = normalize( Σ ρⱼ · W_dec[j] )          # top-50 绝对相关特征加权
```

**注意**：SAE 输入使用原始激活（不做 StandardScaler），因为 threshold、W_enc、b_enc 均在原始分布下训练。

**方法性质**：半监督——VAD 标签参与了方向选取，说明 SAE 特征空间与 VAD 结构**相容**，但不能作为模型自发发现 VAD 结构的无监督证据。

## 结论

1. **PC1 ≈ 效价，PC2 ≈ 唤醒度**，Gemma 2 2B 最大方差方向与人类情绪两个基本维度自发对齐，与原论文一致。

2. **支配度是 PCA 的盲区**：10 个 PC 中没有任何一个能干净对齐支配度，说明支配度信息分散在低方差方向，PCA 无法捕捉。

3. **SAE 通过稀疏分解能构造支配度方向**（ρ=0.742），原因是 SAE 将 superposition 拆解为更单义的特征，再通过 VAD 标签筛选，能找到 PCA 看不到的方向。

4. **encode-then-mean 修正有效**：激活特征数从 13.5 恢复到 31.3，更准确地反映了稀疏结构；VAD 相关系数整体小幅提升，结果稳健。

---

## 局限 & 下一步

- SAE 方向构造是半监督的，不能作为模型"自发学习"VAD 结构的证据
- 叙述数量（20条）仍远少于原论文（100主题 × 12条 = 1200条），均值向量仍有噪声
- 使用层固定为第 25 层，未做层级扫描验证该层是情绪表征最强的层
- 14 个情绪未匹配 NRC-VAD（多词短语），被排除在 VAD 分析之外
- 下一步：用 SAE 合成方向做 Steering 实验，验证因果效应
